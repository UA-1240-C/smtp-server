Detailed Description

Interface for a mail database.

The IMailDB class provides methods for connecting, user management, and email management within a mail database. This interface must be implemented by any class that manages email and user data storage.

If should_retrieve_all is false, retrieves emails that have not yet been marked as received. If should_retrieve_all is true, retrieves all emails for the specified user. If there are no emails for the specified user returns empty vector
