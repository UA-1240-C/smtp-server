std::future< void > TlsSocketManager::PerformTlsHandshake ( boost::asio::ssl::stream_base::handshake_type  handshake_type) const

Asynchronously performs a TLS handshake on the underlying socket.

This function initiates an asynchronous TLS handshake operation, either for the client or server side, depending on the handshake type. The result of the handshake is returned through a future, which will either complete successfully or contain an exception in case of an error.

Parameters
handshake_typeSpecifies whether the handshake is for the client or server side, using boost::asio::ssl::stream_base::handshake_type.
Returns
A std::future<void> which will complete when the handshake operation is done. If the handshake fails, the future will contain an exception describing the error.
Exceptions
std::runtime_errorThrown if the handshake operation fails with a specific error.
