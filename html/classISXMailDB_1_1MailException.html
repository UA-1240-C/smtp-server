Detailed Description

Custom exception class for handling mail-related errors.

The MailException class is a custom exception used to represent errors that occur during mail operations within the ISXMailDB namespace. It inherits from std::exception and provides additional functionality to carry an error message.

Note
This class is intended to be used to signal issues specific to mail database operations, such as failures in database queries, invalid data, or connection problems.
